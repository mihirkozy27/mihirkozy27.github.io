EECS 470 Final Project: Out-of-Order Processor

In a group of five students, we were able to design, test, and synthesize a processor with MIPS R10K style (true) register renaming using Synopsys VCS. Our processor achieved an average cycles per instruction (CPI) of 1.82 and clock period of 19.5 ns with a 3-way superscalar design. We implemented N-way superscalar, store to load forwarding, instruction prefetcher, non-blocking instruction and data caches with load and store coalescing, 2-bit bimodal predictor, branch target buffer, and pipelined functional units. This project is a private school project so unfortunately I cannot provide a link to the code due to honor code reasons.

Brushed Motor Controller

Developed brushed motor controller for Michigan Mars Rover team to drive up to 20 amp brushed motors. Used H-Bridge schematic design and STM32 microcontroller with CAN bus capabilities to provide logic controls to gate drivers. Helped MRover reach 4th place finish in 2023 University Rover Challenge

Brushed Motor Controller

Built an autonomous and Nintendo 64 controller driveable robot in a group of 4 to launch ping pong balls up to 12 feet with 3 inch landing precision to let players practice returning variable velocity and spin serves without a partner. We created a custom bit-banged GPIO ARM assembly routine to emulate N64 proprietary protocol to communicate to controller. We derived odometry from 9 DoF IMU (I2C), edge detecting ultrasonics sensors (PWM), and motor controller data. We demonstrated this project at the Winter 23 EECS 373 expo in April of 2023!
